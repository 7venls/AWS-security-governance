# AWS とはなにか？

・AWS（Amazon Web Services）は、Amazon が提供するクラウドサービスの総称である。

・従来のサーバを自分で用意する「オンプレミス」と違い、AWS では必要なときに必要な分だけ使えるサーバやストレージをネット経由で利用できる。

### メリット

・初期投資が不要（サーバを買わなくてOK）

・必要に応じてリソースを増減可能

・世界中の拠点でサービス提供可能

・セキュリティやバックアップが整っている

#### AWS は「サービスやネットワークの箱を借りる」イメージ

# リージョン（Region）とアベイラビリティゾーン（AZ）

AWS は世界中にデータセンターを持っている。

### リージョン（Region）

・地理的に分かれた大きなデータセンターの集まり

例：東京（ap-northeast-1）、米国バージニア（us-east-1）

・リージョンごとにサービスや料金が少し異なることもある。

### アベイラビリティゾーン（AZ：Availability Zone）

・1つのリージョン内にある独立したデータセンター

・複数のAZを使うことで、障害が起きてもサービスが止まらない構成が作れる

#### イメージ図

Region（東京リージョン）

 ├─ AZ1（データセンター1）
 
 ├─ AZ2（データセンター2）
 
 └─ AZ3（データセンター3）
