# EC2

・Elastic Compute Cloudの略で、AWS上の「仮想サーバー」。

・物理サーバーではなく、クラウド上で自由に作ったり消したりできるサーバー。

・特徴：起動時間で課金、自由にＯＳやスペックを選べる。

# EC2の主要用語

### AMI(Amazon Machine Image)

・サーバーのテンプレート。

・例：Amazon Linux2023、Ubuntu 22.04

・OSや初期設定が含まれている。

### インスタンスタイプ

・サーバーの性能を決める（ＣＰＵ、メモリ）。

・例：t2.micro(小規模用）、t3.large(中規模用)。

・「スペック」を求めるイメージ。

### EBS(Elastic Block Store)

・EC2の「ハードディスク」の役割。

・SSDのようにデータを保存、EC2を停止してもデータ残る。
